# PlanetPic

- Image uploading project developed with VueJS.

## API Setup
```
cd api
npm init
```

## API Init
```
cd api 
npm start
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
